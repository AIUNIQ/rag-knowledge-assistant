# Examples

This folder contains representative examples of how the system interacts with documents, users, and AI models.

## Example Workflow

User Question:

"What is the cancellation policy?"

System Process:

1. Generate query embedding.
2. Search vector database.
3. Retrieve relevant knowledge chunks.
4. Build context package.
5. Generate grounded response.

Response:

"The cancellation policy allows cancellation within the defined service window according to company policy."
