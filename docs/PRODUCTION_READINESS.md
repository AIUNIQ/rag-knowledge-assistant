# Production Readiness Checklist

## Data Quality

- Source documents reviewed
- Duplicate content removed
- Outdated content archived
- Metadata applied
- Retrieval tests completed

## Retrieval Quality

- Top-K retrieval tested
- Irrelevant chunks reduced
- Query variations tested
- Edge cases reviewed
- Low-confidence behavior defined

## AI Behavior

- System prompt tested
- Guardrails enforced
- Hallucination checks completed
- Clarifying question behavior tested
- Escalation path documented

## Infrastructure

- Vector database configured
- API endpoints tested
- Docker deployment prepared
- Logs enabled
- Monitoring defined

## Security

- Sensitive documents protected
- Access control considered
- API keys secured
- Environment variables used
