# Project Highlights

## Overview

RAG Knowledge Assistant architecture designed to retrieve accurate information from documents, knowledge bases, transcripts, spreadsheets, and operational data.

The system demonstrates how AI assistants can generate grounded answers using embeddings, vector databases, semantic search, and retrieval-augmented generation.

## Technologies

- OpenAI
- GPT-4o
- Python
- FastAPI
- Embeddings
- Pinecone
- ChromaDB
- Qdrant-compatible design
- Docker
- Ubuntu Linux
